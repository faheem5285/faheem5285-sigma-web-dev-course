CSS Units Demo (Arranged by Use)

1. Commonly Used Units

px (pixels)
✅ When to use: For precise, fixed UI elements (icons, borders, shadows).
❌ When not: For responsive typography/layouts (not scalable).
20px text
rem (root em)
✅ When to use: For consistent typography and spacing (scales with root).
❌ When not: If you need scaling relative to parent only.
2rem text
em (element relative)
✅ When to use: For spacing/margins that should scale with element font size.
❌ When not: In deep nesting (values can multiply unexpectedly).
2em text
% (percentage)
✅ When to use: For fluid layouts relative to parent size.
❌ When not: If parent has no defined size (becomes unpredictable).
50% width box
vw (viewport width)
✅ When to use: Responsive font sizes, full-width sections.
❌ When not: Very small/large screens (can make text unreadable).
5vw text
vh (viewport height)
✅ When to use: Fullscreen hero sections or modals.
❌ When not: Mobile browsers (100vh may ignore top/bottom bars).
20vh box

2. Less Common but Useful

ch (character width)
✅ When to use: Inputs/code blocks sized by character count.
❌ When not: For non-monospace fonts (varies too much).
20ch wide
ex (x-height)
✅ When to use: Rarely, for fine-tuned text alignment.
❌ When not: Inconsistent across fonts/browsers.
2ex text

3. Grid & Layout Units

fr (fractional unit)
✅ When to use: In CSS Grid for proportional layouts.
❌ When not: Outside grid (not supported elsewhere).
1fr
2fr

4. Rare / Print-Oriented Units

pt (points)
✅ When to use: Print stylesheets.
❌ When not: On screens (inconsistent).
20pt text
cm (centimeters)
✅ When to use: Print layouts.
❌ When not: Web screens (no consistency).
5cm wide
mm (millimeters)
✅ When to use: Print layouts.
❌ When not: Web screens.
50mm wide
in (inches)
✅ When to use: Print (physical size).
❌ When not: Web.
2in wide
pc (picas)
✅ When to use: Historical print media.
❌ When not: Web.
2pc text
cap (capital height)
✅ When to use: Typography-sensitive designs.
❌ When not: Browser support is still limited.
10cap text
ic (ideographic character unit)
✅ When to use: East Asian layouts (Chinese, Japanese).
❌ When not: Latin-based designs (unnecessary).
10ic wide
