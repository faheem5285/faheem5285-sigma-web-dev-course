Here we will Learn About em And rem.

This is parent tag. Here Font will be remain same as we add in class i mean if we add font 16px here will remain 16

Child tag. Here font will be change or you can say that font will inherit from parent tag if we use em size unit in font size of this child font size. if we give font size 16 px in parent tag and we give 2em in child this will increase with 2*16 and if we give 3em this will be 3*16 and also if we give 0.3*16 also show small

Here we will see To rem

Child tag. Here font will also change, but unlike em, rem is based on the root (html) font size, not parent. For example, if the root html has font-size: 16px; - 2rem = 2 * 16 = 32px - 3rem = 3 * 16 = 48px - 0.3rem = 0.3 * 16 = 4.8px (smaller) So rem always refers to root font size and ignores parent inheritance.
